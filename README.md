https://visualstudio.microsoft.com/

4. เปิด vs code (ด้วยสิทธิ์ admin) แล้วไปที่ โปรเจค
4.1 npm i

พอลง environment ได้แล้ว ให้ทดสอบ run node
nodejs capture.js
